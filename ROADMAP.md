# teachAI 1ヶ月商業化ロードマップ

> **目標**: 1ヶ月でMVP→有料課金ユーザー獲得までのプロダクト体験を完成させる

---

## 現状分析

### 強み
- **独自の学習メソッド**: "教えて学ぶ" ピアチュータリングは学術的根拠あり (Learning by Teaching)
- **高度なAIエンジン**: v3 SOLO Taxonomy採点、RQS、KB検出、Eloレーティング
- **マルチプロバイダー**: Claude / GPT / Gemini / Bedrock 対応 → ベンダーロックイン回避
- **豊富な入力対応**: YouTube / Web / PDF / DOCX / PPTX / 画像 / テキスト
- **キャラクターシステム**: 学習パートナーとしてのAIキャラ → 情緒的つながり

### 課題（商業化ブロッカー）
| 課題 | 影響 | 優先度 |
|------|------|--------|
| ランディングページ不在 | 新規ユーザーの獲得不可 | 🔴 Critical |
| オンボーディング未実装 | 初回離脱率が高い | 🔴 Critical |
| リテンション機構なし | 継続利用されない | 🔴 Critical |
| 課金システム未実装 | 収益ゼロ | 🟡 High |
| PWA/OG/SEO未整備 | 共有・発見されない | 🟡 High |
| ソーシャル共有なし | バイラル成長不可 | 🟡 High |
| アナリティクス未導入 | データ駆動の改善不可 | 🟢 Medium |
