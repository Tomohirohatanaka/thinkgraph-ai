# /kpi-status

KPI ダッシュボードを更新し、現状を報告します。

## 実行内容

1. Vercel Analytics / Supabase からデータ取得
2. Stripe 売上を取り込み(`bookkeeping-monthly` 経由)
3. KPI ダッシュボードテンプレに転記
4. 異常値・トレンド変化を明示
5. 月次の場合: `accountant` も呼んで月次クローズと連動

## KPI(§25 準拠)

- 獲得: 新規登録数(任意指標)
- 活性化: 初回検算セッション完了率
- 継続: 7日継続 / 30日継続(中毒性ではなく "戻ってきた" の指標)
- 収益: MRR + ライフタイム累計
- 知財: 出願件数 / 進捗
- 財務: 月次収支 / L1 黒字化進捗

## 使い方

```
/kpi-status [月次/週次/任意期間]
```

## 重要
