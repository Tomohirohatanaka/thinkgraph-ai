# 営業フォローアップ草案

## 基本姿勢

- **断るべきものは断る** — §3.4#7 強制利用販売、#8 子供向けは無条件で辞退
- **押し売り禁止** — 「今だけ」「特別に」を使わない
- **代替案を添える** — 自分で対応しない場合は、別の良い選択肢を紹介する誠実さ
- **個人 OR 法人ライト** — 個人/小規模チーム向けの利用が中心

## シナリオ別テンプレ

### A. 個人ユーザー問い合わせ(機能質問)
```
件名: <相手の質問の要点> についてお答えします

<相手のお名前> 様

ご質問ありがとうございます。

お問い合わせいただいた <質問内容> についてですが、
<事実ベースの回答>。

teachAI は「教える側として説明する」体験を中心にしており、
<相手が達成したいこと> には <具体的な使い方> が合うと思います。

もし合わなさそうでしたら、<代替の選択肢> もあります(無理にお勧めしません)。

質問が続く場合はお気軽にお返事ください。
```

### B. 法人(企業)からの問い合わせ — 個人利用前提なら受ける
```
件名: 法人内でのご利用について

<担当者名> 様

お問い合わせありがとうございます。

teachAI は現状、個人での "検算" 体験を主目的としており、
法人向けの一括導入・研修義務化のような形式は提供していません。

社員の方が個人の自己研鑽として任意に使われる場合は、
個人プランを各自でご利用いただく形が最も適切と考えています。

将来的に法人向けプランを検討する場合も、
「全員が必ず使う」設計ではなく、「使いたい人が使う」設計を保ちます。

ご期待に沿えなかった場合はご容赦ください。
```

### C. 教育機関(中高大)からの問い合わせ
```
件名: 教育機関でのご利用について

<担当者名> 様

お問い合わせありがとうございます。

teachAI は現時点で 18 歳未満の利用を想定しておらず、
中学・高校での導入はご遠慮させていただいています。

大学・大学院・社会人スクールの場合は、
個人での自主利用としてご活用いただけます。
ただし「単位連動」「課題提出義務化」のような使い方は、
弊サービスの設計思想に合わないため、お控えください。

該当用途で関連サービスをお探しでしたら、
<代替サービス候補> をご検討いただくと良いかもしれません。
```

### D. 中高生(本人)からの問い合わせ
```
件名: ご利用について

<本人名> 様

teachAI に興味を持っていただきありがとうございます。

申し訳ないのですが、teachAI は 18 歳以上の方を対象としています。
高校卒業後、機会がありましたらまたお声がけください。

学習に関する別のリソースとしては、
<無料の公的リソース、Khan Academy など> がおすすめです。
```

### E. パートナーシップの誘い
```
件名: ご提案ありがとうございます

<担当者名> 様

ご提案ありがとうございます。
内容を拝見しました。

teachAI は「人間の思考力を取り戻す」を最優先に据えており、
<相手の提案内容> は <整合する/しない> と考えています。

<整合する場合>
具体的な進め方について、<次のステップ> でご相談させてください。

<整合しない場合>
今回はお受けできないのですが、ご提案の方向性自体は理解しております。
<代替の関わり方> でしたら検討可能です。
```

## NG パターン

- 「無料体験」と称した解約困難化導線
- 「今だけ X% OFF」
- 教育機関の児童利用に「条件付きで OK」と返事
- 法人案件で「全社員」を前提に話を進める
- 競合の悪口

## 作成プロセス

1. 相手のシナリオ判定(A-E)
2. テンプレ選択
3. 個別事情を反映してドラフト
4. NG パターンと照合
5. 30 秒で読める長さに調整

## 重要

「売上のために北極星を曲げない」が大原則。断るときは丁重に、しかし曖昧にしない。
